단체미로 공략

단체미로 관련 자료를 순서대로 정리한 안내 페이지입니다. 통로별 이동 경로, 용문 위치, 시작·도착 구조를 한 번에 확인할 수 있게 구성했습니다.

총 7개 자료공지에서 바로 진입 가능한 이미지형 공략
모바일 대응휴대폰에서도 세로로 순서대로 보기 쉽게 구성
이미지 확대 지원이미지를 누르면 크게 열어 세부 경로 확인 가능

빠른 이동

원하는 구간으로 바로 이동할 수 있게 순서별 링크를 넣었습니다.

01. 4번 통로입구에서 출구 방향 경로 확인 02. 용문 위치좌표와 핵심 위치 참고용 지도 03. 1번 통로입구 기준 1번 이동 루트 04. 중앙 이동 경로중간 연결 구간 참고용 05. 2번 통로입구 기준 2번 이동 루트 06. 3번 통로입구 기준 3번 이동 루트 07. 시작 · 도착 구조미로 기본 배치와 목표 지점 확인

공략 자료

위에서 아래로 보면 자연스럽게 흐름을 파악할 수 있도록 배치했습니다.

01

4번 통로

4번 통로 기준 이동 경로를 표시한 자료입니다. 출구 방향까지 이어지는 흐름을 먼저 익힐 때 보기 좋습니다.

02

용문 위치

좌표별 핵심 위치와 용문 위치를 확인할 수 있는 참고 지도입니다. 동선 파악 전 전체 위치를 먼저 볼 때 유용합니다.

03

1번 통로

1번 통로 기준 이동 경로를 정리한 자료입니다. 입구와 출구 방향을 연결해서 보기에 편한 순서로 배치했습니다.

04

중앙 이동 경로

중간 연결 지점 중심으로 이어지는 경로를 표시한 자료입니다. 통로 간 이어지는 흐름을 볼 때 참고하면 좋습니다.

05

2번 통로

2번 통로 이동 경로를 정리한 자료입니다. 다른 통로와 헷갈리지 않게 순서형으로 확인할 수 있습니다.

06

3번 통로

3번 통로 기준으로 입구에서 진행하는 동선을 정리한 자료입니다. 좌측 방향 진입 흐름을 확인하기 좋습니다.

07

시작 · 도착 구조

단체미로의 시작 지점과 도착 지점을 확인할 수 있는 전체 구조 자료입니다. 처음 보는 유저에게 기본 배치를 설명할 때 적합합니다.

이 페이지는 notices.json에서 직접 연결할 수 있도록 만든 정적 상세 페이지입니다. 기본 경로는 maze-guide.html 기준으로 작성했습니다.
