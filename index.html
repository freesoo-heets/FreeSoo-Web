공지, 공략, 다운로드를
한 화면에서 정리하는 메인 허브

기존 한 페이지 허브 구조는 유지하면서, 공지 자동 출력과 공략 확장성을 강화한 맞춤형 통합 버전입니다. 앞으로는 이미지와 JSON만 추가해도 홈과 공략집이 같이 확장되도록 설계했습니다.

공략집 바로가기 최신 공지 보기 무공설명서 열기
notices.json 자동 출력 홈 공지와 이벤트를 데이터 기반으로 렌더링해 HTML 수정을 최소화합니다.
guides.json 공략 연동 대표 공략 2개를 메인에서 자동 노출하고 전체 공략집으로 자연스럽게 연결합니다.
게임 UI 스타일 강화 먹선+주황 발광 분위기와 카드형 인터랙션으로 포털 느낌보다 게임 허브 느낌을 더 살렸습니다.
운영 방식 공지 추가는 notices.json, 공략 추가는 guides.json만 수정하면 됩니다.
관리자 페이지 역할 정적 사이트 특성상 서버에 직접 저장되지는 않지만, JSON 파일을 자동 생성·다운로드하는 도우미로 사용할 수 있습니다.
권장 폴더 구조 assets/guides/공략ID/ 안에 이미지를 넣고, JSON에는 제목·설명·순서·썸네일만 기록하는 방식이 가장 안정적입니다.

운영 팁

권장 파일 구조

  • assets/guides/maze/처럼 공략별 폴더를 고정합니다.
  • guides.json에서 제목, 설명, 순서, 썸네일을 관리합니다.
  • notices.json은 홈 공지와 이벤트 카드에서 공통 사용합니다.
  • 이미지가 많아지면 JPG/WebP 혼용으로 용량을 줄이는 편이 유리합니다.
정적 사이트라서 관리자 페이지가 서버에 직접 저장할 수는 없습니다. 대신 JSON 파일을 자동 생성해서 다운로드 후 GitHub에 업로드하는 흐름으로 쓰면 가장 안정적입니다.
